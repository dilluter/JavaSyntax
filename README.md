Repositório de Estudos Java
Este repositório contém uma coleção de códigos e exemplos desenvolvidos durante meus estudos da linguagem de programação Java. O objetivo é registrar e documentar meu progresso, cobrindo desde conceitos básicos da sintaxe até tópicos mais avançados, como o uso de anotações e a introdução a frameworks populares.

Tópicos Abordados
Os arquivos estão organizados por assunto, cobrindo as seguintes áreas:

1. Fundamentos da Linguagem (Commit: BASICO)
EstruturaSequencial.java - Conceitos iniciais, processamento de dados e operações sequenciais.

EstruturaCondicional.java - Exemplos de uso de if-else, else if e switch-case.

EstruturaRepeticao.java - Exemplos com laços de repetição while, do-while e for.

Vetores.java - Declaração, inicialização e manipulação de arrays unidimensionais.

Listas.java - Uso da interface List e da classe ArrayList para coleções dinâmicas.

DataHoraJava.java - Manipulação de datas e horas com a API java.time (LocalDate, LocalDateTime).

Wrappers.java - Conceito e uso de classes Wrapper para tipos primitivos (ex: Integer, Double).

2. Programação Orientada a Objetos (POO)
EstruturaComportamentoDeClasse.java - Fundamentos de classes, atributos, métodos, construtores e encapsulamento.

ProblemaPOO.java - Resolução de um problema prático aplicando os conceitos de POO.

3. Tópicos Avançados e Anotações (Commit: Anotações)
DemonstracaoAnotacoes.java - Um guia detalhado, dentro do próprio código, sobre as anotações padrão do Java, como @Override, @Deprecated, @SuppressWarnings, @FunctionalInterface e @SafeVarargs.

DemonstracaoFrameworks.java - Um arquivo conceitual que explica, com exemplos comentados, o uso de anotações de frameworks populares como Spring (@RestController, @Autowired), JPA/Hibernate (@Entity, @Id) e JUnit (@Test).

EstudoJavaCompleto.java - Um arquivo consolidado com múltiplos exemplos para uma revisão completa dos tópicos.

Como Usar
Pré-requisitos
É necessário ter o Java Development Kit (JDK) instalado na sua máquina.

Compilando e Executando
A maioria dos arquivos pode ser compilada e executada diretamente pelo terminal.

Navegue até a pasta onde o arquivo .java está localizado.

Compile o arquivo:

Bash

javac NomeDoArquivo.java
Execute o arquivo compilado:

Bash

java NomeDoArquivo
Substitua NomeDoArquivo pelo nome do arquivo que deseja testar (ex: EstruturaCondicional).

Sobre Frameworks
O arquivo DemonstracaoFrameworks.java é apenas para estudo e não pode ser compilado da maneira acima. Ele depende de bibliotecas externas (Spring, JPA, etc.) que precisam ser gerenciadas por uma ferramenta de build como Maven ou Gradle em um projeto completo.
